## Claude Platform on AWSが一般提供開始
* 2026-05-11T16:03:26.000000Z claudeai
* https://twitter.com/claudeai/status/2053868592286822443
* https://claude.com/blog/claude-platform-on-aws

「Claude Platform on AWS」は、**ネイティブ Claude API のフル機能セット**を AWS 経由で提供するもので、新機能やベータも **ネイティブ API と同日にリリース** されます(day-one parity)。認証は AWS IAM、監査は CloudTrail、課金は AWS の請求書経由で AWS のコミットメント消化にも使えます。

ざっくり、表で整理するとこんな感じだそうです。

| 項目 | Anthropic 直契約 (API) | Claude Platform on AWS | Amazon Bedrock |
|---|---|---|---|
| ネイティブ API 機能 day-one | ✅ | ✅ | ❌(遅れる場合あり) |
| データ処理境界 | Anthropic | Anthropic(AWS 境界外) | AWS 内 |
| 認証/課金 | Anthropic Console | AWS IAM / AWS 請求 | AWS IAM / AWS 請求 |
| claude.ai, Pro/Max など | ✅ | ❌(対象外) | ❌ |
